# Atividades Individuais - POO em Java
**Melissa Lima**
## Exercícios
### 1. Simulador de Caixa Eletrônico
 * **Foco:** Lógica procedural (switch, do-while).
 * **Regra:** Controle de limites de saques diários e validação de valores máximos por operação.
### 2. Conta Bancária
 * **Foco:** Fundamentos de POO (Classes e Objetos).
 * **Destaque:** Uso de construtores para garantir bônus de boas-vindas e métodos encapsulados de saque e depósito.
### 3. E-commerce
 * **Foco:** Relacionamentos entre objetos (Cliente, Pedido e ItemPedido).
 * **Destaque:** Cálculo automático de frete baseado no valor total do pedido.
### 4. Sistema de Veículos
 * **Foco:** Interfaces, Polimorfismo e Sealed Classes (Java 17).
 * **Regras:** Tributação de IPVA com isenção por idade e cálculo de locação com taxa de sobrecarga para carga pesada.
## Tecnologias
 * **Java 17**
 * **IntelliJ IDEA**
 * **Git**
